Gráficos Dinámicos — Visualizar para Decidir

Un gráfico dinámico está vinculado a una tabla dinámica. Cuando cambias los filtros de la tabla, el gráfico se actualiza automáticamente.

Cómo crear: Clic en la tabla dinámica → pestaña "Analizar" → "Gráfico dinámico" → elige el tipo.

Tipos de gráfico según el análisis:

TipoUsar cuando...
Barras/ColumnasComparar categorías (vendedores, ciudades)
LíneasVer tendencias en el tiempo (ventas por mes)
Circular (torta)Mostrar participaciones porcentuales
ComboComparar dos métricas con ejes distintos

Conexión automática: Si filtras la tabla dinámica por mes "Enero", el gráfico cambia solo para mostrar solo enero. No hay que hacer nada más.

Buenas Prácticas con Gráficos Ejecutivos

Principios de visualización:

  • Un gráfico = una sola idea. No sobrecargues
  • Siempre incluye título descriptivo (¿qué muestra?)
  • Etiquetas de datos visibles para valores clave
  • Eliminar cuadrículas y bordes innecesarios
  • Paleta de colores consistente (2-3 colores max)
  • Si el gráfico necesita mucha explicación, es mala

Errores a evitar:

  • Gráficos en 3D (distorsionan las proporciones)
  • Demasiadas categorías en un torta (máximo 5-6)
  • Colores de semáforo para datos que no tienen buen/malo
  • Escalas que no empiezan en cero (engañan)
  • Gráficos sin título ni etiqueta de ejes
Para presentar al gerente: Gráficos simples, claros, con un color de destaque para el dato más importante. El gerente tiene 30 segundos para entenderlo.

Slicers (Segmentadores) y Líneas de Tiempo

Slicers — Botones de Filtro Visual

Son botones que aparecen en la hoja. Un clic filtra la tabla dinámica y el gráfico dinámico simultáneamente.

Insertar: Clic en tabla dinámica → "Analizar" → "Insertar segmentación de datos" → seleccionar campos.

Conectar a múltiples tablas:

Si tienes 3 tablas dinámicas, puedes conectar un slicer a todas. Clic derecho en el slicer → "Conexiones de informe" → selecciona todas las tablas. Un clic = todas se filtran.

Líneas de Tiempo — Filtro de Fechas

Similar al slicer pero específico para fechas. Permite deslizar para seleccionar rangos de tiempo.

Insertar: "Analizar" → "Insertar línea de tiempo" → selecciona la columna de fecha. Funciona solo si la columna es fecha real (no texto).

Niveles disponibles:

  • Días: para análisis diario
  • Meses: el más usado en empresas
  • Trimestres: para informes trimestrales
  • Años: para visión estratégica

Power Query — ¿Qué es y Para Qué Sirve?

Power Query es una herramienta integrada en Excel para importar, limpiar y transformar datos desde múltiples fuentes, de forma automática y repetible.

¿De dónde puede importar datos?

  • Otros archivos de Excel
  • Archivos CSV o TXT
  • Bases de datos (Access, SQL Server)
  • Páginas web (tablas HTML)
  • Carpetas completas de archivos
  • Google Sheets y SharePoint

¿Qué transformaciones puede hacer?

  • Quitar columnas innecesarias
  • Limpiar espacios y cambiar tipos de datos
  • Separar columna por delimitador
  • Combinar (unir) múltiples tablas
  • Calcular nuevas columnas
  • Filtrar filas por condición
Lo mejor: Todo esto se graba como pasos. La próxima vez que llegue el archivo actualizado, Power Query aplica los mismos pasos automáticamente en segundos.

Power Query — Flujo de Trabajo

📂

1. OBTENER DATOS

Datos → Obtener datos → Elige la fuente (Excel, CSV, web). Power Query se abre con los datos crudos.

🔧

2. TRANSFORMAR

En el editor de Power Query: quita columnas, cambia tipos, limpia datos, filtra, combina. Cada acción se registra como un "Paso aplicado".

✅

3. CARGAR

"Cerrar y cargar" → Los datos transformados aparecen en Excel como una Tabla. Lista para tablas dinámicas.

🔄

4. ACTUALIZAR

Cuando llegue el archivo nuevo del mes, clic derecho en la tabla → "Actualizar". Power Query repite todos los pasos automáticamente.

Caso real: El sistema contable exporta un .CSV cada mes. Antes: 2 horas de limpieza manual. Con Power Query: 10 segundos de actualización.
