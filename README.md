# Curso Intensivo de Ofimática 2026

Bienvenido a la plataforma web del **Curso Intensivo de Ofimática** para Carreras Administrativas de UniSalamanca. Esta herramienta ha sido diseñada para reemplazar presentaciones tradicionales por una experiencia de aprendizaje interactiva y fluida.

## 🚀 Cómo iniciar
1. Abre el archivo `index.html` en cualquier navegador (Chrome, Edge, Firefox).
2. Navega por los días del curso usando la barra lateral izquierda.
3. Al final de cada sesión, encontrarás un **Quiz interactivo** para validar conocimientos.

## 📁 Estructura del Proyecto
- `index.html`: Estructura principal de la plataforma.
- `style.css`: Diseño premium con glassmorphism y estilos responsivos.
- `app.js`: Lógica de navegación y sistema de evaluación.
- `data.js`: Contenido completo del curso (objetivos, temas, actividades).
- `assets/`: Imágenes y recursos visuales.
- `recursos/`: Contiene el PDF original del programa.
