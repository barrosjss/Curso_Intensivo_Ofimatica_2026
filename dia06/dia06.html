¿Qué es una Macro?

Una macro es una secuencia de acciones grabadas que Excel puede repetir automáticamente con un solo clic o atajo de teclado.

Analogía: Como grabar un video de lo que haces en Excel. Cuando lo reproduces, Excel hace exactamente lo mismo en segundos.

¿Para qué se usan en empresas?

  • Dar formato a reportes mensuales automáticamente
  • Limpiar datos importados del sistema
  • Generar un reporte en PDF con un botón
  • Enviar correos desde Excel (Outlook)
  • Actualizar y imprimir múltiples hojas

Tipos de macros que aprenderemos:

Macros Grabadas

Graba tus acciones en Excel. No requiere programar. Para tareas repetitivas simples.

Macros con VBA básico

Un poco de código para hacer la macro más inteligente. Veremos lo esencial para el auxiliar.

Prerrequisito: Activa la pestaña "Desarrollador" en Excel: Archivo → Opciones → Personalizar cinta → Activar "Desarrollador".

Grabar una Macro — Paso a Paso

  • Pestaña "Desarrollador" → "Grabar macro"
  • Asigna nombre (sin espacios: "FormatoReporte"), atajo (Ctrl+Shift+R) y descripción
  • Elige "Este libro" como ubicación
  • Clic en Aceptar — ¡ya está grabando!
  • Ejecuta las acciones que quieres automatizar
  • "Detener grabación" cuando termines
  • Prueba: presiona Ctrl+Shift+R → las acciones se repiten

Ejemplo — Macro "FormatoReporte":

Acciones grabadas:

' Lo que hace la macro automáticamente:
Sub FormatoReporte()
  ' Pone negrita a la fila 1
  Rows("1:1").Font.Bold = True
  ' Fondo azul marino en encabezados
  Rows("1:1").Interior.Color = RGB(0,70,127)
  ' Autoajusta columnas
  Cells.EntireColumn.AutoFit
End Sub
Guardar como: Archivo con macros debe guardarse en formato .xlsm (no .xlsx). Excel te preguntará automáticamente.

VBA Básico — Solo lo Esencial

No necesitas ser programador. Con estos 5 conceptos básicos puedes hacer macros útiles para el trabajo administrativo.

' Estructura de una macro
Sub NombreMacro()
  ' Aquí van las instrucciones
End Sub

' Seleccionar una celda
Range("A1").Select

' Escribir un valor
Range("A1").Value = "Total:"

' Hacer negrita
Range("A1").Font.Bold = True

' Mostrar mensaje
MsgBox "¡Reporte generado!"
' Bucle — repetir acción
Dim i As Integer
For i = 1 To 10
  Cells(i, 1).Value = i * 100
Next i

' Condición — si/entonces
If Range("B2").Value > 1000000 Then
  Range("B2").Interior.Color = RGB(0,200,0)
Else
  Range("B2").Interior.Color = RGB(200,0,0)
End If
Editar macros: Desarrollador → Visual Basic → busca el módulo con tu macro. El editor resalta el código por colores.

Dashboard Ejecutivo — ¿Qué es un KPI?

KPI = Key Performance Indicator (Indicador Clave de Rendimiento). Son las métricas más importantes para tomar decisiones.

KPIs para un auxiliar administrativo:

  • Total de ingresos / ventas del período
  • Número de clientes activos vs inactivos
  • Días promedio de cartera (cobro)
  • Porcentaje de pedidos entregados a tiempo
  • Costo operativo como % de ventas

Principios de un buen dashboard:

  • Máximo 5-7 KPIs en la pantalla principal
  • El dato más importante: más grande y prominente
  • Colores con semántica: verde=bien, rojo=alerta
  • Todo debe verse sin scroll
  • Actualizable con un clic

Protección de Datos y Hojas

Proteger una hoja:

  • Pestaña "Revisar" → "Proteger hoja"
  • Establece una contraseña
  • Define qué pueden hacer los usuarios (solo seleccionar, no editar)
  • Aceptar. La hoja queda bloqueada.
Ley 1581 de 2012: Los datos personales (nombres, cédulas, salarios) en archivos de Excel compartidos deben protegerse. No envíes archivos con datos de empleados sin contraseña.

Tipos de protección en Excel:

NivelCómoQué protege
CeldaFormato → Proteger celda (bloqueada)Fórmulas individuales
HojaRevisar → Proteger hojaToda la hoja
LibroRevisar → Proteger libroEstructura de hojas
ArchivoArchivo → Info → Cifrar con contraseñaTodo el archivo
Buena práctica: En dashboards, protege las hojas de datos y fórmulas. Deja visible y sin protección solo la hoja del dashboard. El usuario ve los resultados, no puede romper las fórmulas.
